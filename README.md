# TiSSaT
This repository contains the code corresponding to the paper:  
**TiSSaT: A Time-Scaling State Transformation Approach for Spatio-Temporal Control under Constraints**  
  
This paper is dedicated to construct a time-scaling state transformation (TiSSaT) to facilitate controller design and parameters analysis subject to spatial (full-state constraints and input saturation) and temporal (settling time) constraints.

## Features
- An equivalent evolution from the constrained system in the $t$-domain to an unconstrained one in the $s$-domain
- Range of Prescribed time provides a priori guarantee for the feasible design of TiSSaT-based controllers, and ensures the reliable execution of spatio-temporal tracking tasks under complex constraints
